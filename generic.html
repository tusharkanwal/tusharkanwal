Building the Covid-19 spread tableau dashboard

A Tableau Dashboard created to display the spread of Covid-19 by country along with week by week spread and loss of life
since January 2020 till January 2022.

Introduction:

Covid-19 Pandemic has been raging throughout the world for over 1.5 years, and has affected everyone's life in one way or another. In today's data driven world, this means creation of new data, which can be utilized to gather insights about this pandemic that we have been facing for such an extended period of time.

Combining Tableau's Visualization Prowess with the constantly growing data on Covid-19, in a dashboard will help us gain a bird's-eye view of the spread of the virus as it spread week-by-week along with how much loss of life this terrible pandemic has caused and the most affected countries.

Getting the Data:

The data has been taken from Tableau's Covid-19 Datahub which provides a well maintained and daily updated Covid-19 data. Data from January 2020 to January 2022 was used for this project.

Note: I have slightly pre-processed the data using Excel to make it easier to visualize and remove a few bad/unnecessary data points for the purpose of better visualizations.

Planning the Dashboard:

A key step for building an effective dashboard is planning in advance. A few important metrics I would like this dashboard to reflect are:

  • Total number of cases and lives lost.
  • Countries with the greatest number of cases and lives lost
  • A Map to show increasing number of cases per country.
This may require around 6-7 visualizations, using the most appropriate type for each metric.

Getting started with visualization:

We begin this project by importing the Covid-19 data into Tableau. In this project, we used a web data connector available on data.world. To connect, open tableau and click “Web Data Connector”. A window will open, you need to enter the URL of your data and the data will be successfully imported into Tableau. Once, you have your data in Tableau, you can check for data integrity, change column types, filter out the data that you don't require and do all the prep work that is needed.

Now, we can start with the visualization. The first sheet that we will create will have Total Number of Cases in it, with the report date added to the pages side bar so we are able to see the increased number of total cases as they grew per week.

To do this, the “New Positive Cases Count” from the value field needs to be dragged and dropped to the Text mark type which will display the total number of positive cases. The “report date” field needs to be added to the pages sidebar and changed from year, to weeks view after which a table calculation is added to New Positive Cases Count values as a running sum of values based on the week of report date so, the total number of cases update depending on the week we selected. We can apply formatting as desired and our first sheet is complete.

The second sheet will have the Total Number of Deaths. The process to create this sheet is the same the previous sheet so, we can either follow the process again, or we can right click the first sheet tab and click duplicate which will copy the sheet and we can easily switch out the “New Positive Cases count” with “New Deaths Count” and add the table calculation. Apply formatting as required and another sheet is ready.

The third sheet will visualize the per week growth in the Total Number of Cases. For this, drag the "Report Date" field to the columns shelf and change it from year to weeks. Now, drag “New Positive Cases Count” to the Rows shelf twice as this will generate two separate charts. Apply the same table calculations as we did in the first sheet to both the values in rows and change one of the marks to Line and other to Area. Click on the right side header of either chart and select Dual axis, this will combine the two charts into one. Right click the header on left and hide it as it is not required. After all this is done drag the "Report Date" to pages side bar and change to weeks as done in the previous sheets. Apply formatting of your liking and your sheet is ready.

For the next sheet, we want to create the same lollipop chart as above for Total Number of Deaths. The procedure to create this sheet is the same as above, we just need to replace the Rows shelf with “New Deaths Count” and apply the table calculations, after formatting this sheet will also be ready.

Now, we will be creating a sheet to show the countries which had the Most Number of Cases, this will also carry the "Report Date" in the pages side bar like all the previous sheets we have made so far. In order to make this sheet we need a calculated field for the rankings. To do this right click in the left sidebar and click create calculated field, give it a name and enter the following formula RANK(SUM([People Positive Cases Count]),'desc') and click ok. Drag and drop this new field to the rows shelf. Now we need to add a table calculation to this, select computing using Specific Dimesnions and check the Country Short Name. We need a table calculation on New Positive Cases Count too, set calculation type to running Total SUM and compute using specific dimension and choose week of Report Date.

Press and hold the ctrl key and drag the Rank field to the filters shelf. Click edit filter and select numbers 1 to 10 or as many as you want to show. Add desired formatting and the sheet is complete.

In the next sheet we calculate the countries with Most Number of Deaths, which will be done in the same manner as above, except in place of "New Positive Cases Count" we will use "New Death Count"in its place. Create a new calculated rank field and performed the calculation in the same manner as the previous sheet. After everything is done the sheet will be ready and we can move on to the last sheet.

In the last sheet we will be showing the Spread of Virus per Country and how it grew as weeks passed by. Making this is fairly simple, drag and drop the "Country Short Name" field to the visualization area and Tableau will automatically generate a map and plot all the points on it. Next we want to add the "New positive Cases Count" to the Size mark so we can see which country had more cases, we also need to perform a table calculation on the field in the same manner we did in the previous sheets where we have used the "New Positive Case Count"field. After that, add the "Report Date" to the pages side bar and select the weeks,format as desired and the sheet is ready.

Finally, we can create the dashboard with all the sheets that we have created so far. You can decide if you want to use the titled format or floating windows for arranging the sheets in the dashboard. Remove any unwanted filters/legends and edit the dashboard as desired and The Covid-19 spread Dashboard is ready.
